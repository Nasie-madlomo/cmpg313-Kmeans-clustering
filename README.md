# K-Means Clustering on Dense Network Topology

This project applies K-Means clustering to a dense network of 600+ nodes, each with spatial coordinates and energy levels. Clustering is performed for k=7 and k=3, and results are compared.

## Files
- `kmeans.py` – Python script using scikit-learn's KMeans
- Screenshots of 2D scatter plots and 3D surface plots for both cluster counts

## Requirements
- numpy
- pandas
- matplotlib
- scikit-learn
